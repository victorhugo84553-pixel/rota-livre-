# Rota Livre

Aplicativo gratuito para entregadores da Shopee.

## Objetivo
Ler o PDF do Circuit, escanear os pacotes e mostrar o número da parada automaticamente.

## Versão 1
- Importar PDF do Circuit
- Ler códigos BR
- Escanear QR Code
- Mostrar número da parada
- Funcionar offline
